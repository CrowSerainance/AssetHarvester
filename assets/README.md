# Assets Directory

Place application assets here.

## Required for Build

### icon.ico
Place your application icon here as `icon.ico` for the Windows executable.

Requirements:
- Format: ICO (Windows Icon)
- Recommended sizes: Include 16x16, 32x32, 48x48, 256x256
- Tools to create: 
  - https://icoconvert.com/
  - GIMP (File > Export As > .ico)
  - Photoshop with ICO plugin

If no icon is provided, the build will use the default Python icon.
