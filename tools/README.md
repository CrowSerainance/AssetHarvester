# Tools Directory

This directory contains external tools used by Asset Harvester.

## QuickBMS

Download QuickBMS from: https://aluigi.altervista.org/quickbms.htm

Place `quickbms.exe` in this directory.

## BMS Scripts

Place game-specific BMS scripts in the `scripts/` subdirectory.

### Finding BMS Scripts

- Official QuickBMS scripts: https://aluigi.altervista.org/papers.htm
- Community scripts: Search for "[game name] bms script"
- XeNTaX forum archives

### Common Scripts

| Game | Script Name | Notes |
|------|-------------|-------|

| MU Online | mu_online.bms | BMD archives |
